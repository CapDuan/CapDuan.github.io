Django 实践

2021-09-14 00:59:15

基础命令

项目管理

  • django-admin startproject [project name] | 创建项目
  • django-admin startapp [app name] | 创建应用 |
  • python manage.py makemigrations | 创建数据库迁移文件 |
  • python manage.py migrate | 将生成的迁移文件应用到数据库 |
  • python manage.py createsuperuser | 创建管理员

调试

  • python manage.py runserver | 启动项目，默认监听 8000 端口 |
  • python manage.py runserver 0:8080 | 自定义 ip 和 port |
  • python manage.py sqlmigrate [app name] [0001] | 查看迁移时执行的 sql 指令
  • python manage.py shell | shell |
  • python manage.py test | 执行测试 |
  • python manage.py test [app name] | 测试指定应用 |
  • python manage.py test [appname.tests.test_xxx_xx] | 测试指定文件（不能加 py 后缀） |
  • python manage.py test [--verbosity=2] | Verbosity 决定了将要打印到控制台的通知和调试信息量;0 是无输出，1 是正常输出，2 是详细输出。

要成为一个怎样的人

要成为一个怎样的人，应该说是

“我想成为一个怎么样的人？”
“我该成为一个怎么样的人？”

“想”是自由愿望，是抛开所有束缚的信马由缰

孩提时代，每每在台灯下看着各科的书卷和教辅。
想的是快快长大，可以随意支配自己的时间和精力，可以用“我也没有办法”来搪塞一切问题，而且最重要的他们想什么时候睡觉就可以，累了可以早休息，有球赛可以晚休息，而我只能根据作业的进度来决定自己的睡眠时间(但其实我也没怎么认真写过几年作业，也就初高三认真写过两年)。一秒变大，从此再无作业。但幼稚的脑壳想象不到变成一个成人需要的牺牲。

单身的时候，想要成为一个有对象的人。拇指姑娘四季常伴，这是一个天大的问题。我知道性是一个正常的话题，你也知道性是一个正常的话题，大家都知道性是个正常的话题，恰如吃喝拉撒般。
可是当我们把这个问题从脑壳里准备拿出来，准备说出来，准备写出来的时候，又万般扭捏。
他们说恋爱都始于颜值。审美标准暂且不表，人人心中有杆美的标准秤。但为什么会始于颜值。在我这无非就是内心深处的渴望和冲动。
年纪轻轻，如狼似虎，万般扭捏。

想成为一个经济富足的人，过年回家，亲朋好友无不为了经济而苦恼，而我也是长期深陷经济危机，思考为什么到这个年纪，身家百万的，身家千万的，都有这样的苦恼。感觉大家其实在意的不是绝对的静态量，而是在意一定时间的变化量。
一个是速度，一个是加速度。60迈的速度可以活，120迈也可以活。
但好像大家都会纠结，“下一段路我能不能开到70，你看人家老王，都开到62了”
其实这都是些浅显易懂的道理。你懂我懂大家都懂。但很多问题也出在这里，冒着风险，顶着压力，害着身体。你的车可能这辈子就只能开到70，为什么就不能定速巡航到70然后去认认真真听听歌呢。他们的车凭借着祖祖辈辈的积累，自己的拼搏，随随便便就120。你凭什么觉得你把油门踩进油箱，你就能超过他呢。油门踩进油箱，车毁人亡。

但却是从未想过，我会变得喜欢写作。
写文章是为了讨好别人还是讨好自己。讨好自己是可以把思考具体到文字，让未来回想起，自己的思考是如何，就像把一大堆思绪都简化成为了图像，看到便能会想到。但讨好别人，又是另一种的讨好自己，表达欲望得到了满足，企图用自己的思想去影响别人。脑子里的东西其实说出来就变味道了。别人听到就更变味了。但是我不管，我就是要说，你听不听是你的事情，我说不说是我的事情。

“该”是框架边界，是老祖宗说的方圆
扯点毁三观的。奉行孝道，想还是该。

扯的有点远，其实我想说的，想成为一个怎么样的人这件事，是一个集合，它包括着你的一切自由意志，而该成为一个什么样的人，也是一个集合，他的边界是你所存在的时代，国家，社会，家庭所塑造的所谓责任，所谓梦想，所谓理想，所谓担当。
这两个集合相互博弈，相互竞争，相互妥协。
而其中的交集就是你要成为的人。

后记.
写下这些是想还是该，悲哀的是我觉得这是这个时代塑造的一种表达，你我逃不开浪潮的拍打。
没有什么你以为，无非机缘巧合，无数变量，造就这个世界，这般国家，这般社会，这般家庭，这般的你。这般你以为的一切。
所以，没有所谓自由意志。你有思辨，便不会自由。
结局已定。
